Behaviors Demo

Examples

Elliptical Behaviors for Polymer Components provides deep observable two-way data-binding without imperative wiring as well as implementing dependency injection/service locator for tighter compatibility with the MVC design pattern.

Hello-World

```html <dom-module module id="hello-world"> <style> :host ::content h4{ color:#9C27B0; } </style> <template> <h4 on-click="handleClick">Hello, World</h4> </template> </dom-module> ``` ```js Polymer({ is: 'hello-world', handleClick:function(event){ console.log(event); } }); ```
